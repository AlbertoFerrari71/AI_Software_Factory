# Codex Controlled Code Prompt

Lavora sul branch indicato nel task packet.

Segui obbligatoriamente:

- `README.md`
- `AGENTS.md`
- `docs/10_ROADMAP.md`
- `docs/05_SECURITY_MODEL.md`
- il Codex Task Packet allegato

## Regole

- Modifica solo i file elencati come modificabili.
- Non toccare i file vietati.
- Non fare commit.
- Non fare push.
- Non fare merge.
- Non introdurre dipendenze senza motivazione.
- Aggiorna la documentazione se cambia comportamento.
- Esegui i test indicati.
- Se i test falliscono, fermati e spiega.

## Output finale richiesto

1. File modificati.
2. Riepilogo diff.
3. Test eseguiti.
4. Test falliti o non eseguiti.
5. Rischi residui.
6. Rollback consigliato.
7. Prossimo passo.
