- local-first supervised operation;
- human-gated publication path;
- Bridge report Markdown plus JSON sidecar;

- operator status CLI;
- GPT reviewer packet;
- draft Codex prompt;
- publish readiness semaphore;

- risk classifier golden eval with zero-downgrade policy;
- optional local and CI secret scanning foundation;
- Windows PowerShell runner CI coverage.

## What ASF V1 does not do

- no web UI;
- no scheduler;
- no autonomous Codex;
- no mandatory live AI calls;
- no automatic publish;
- no automatic commit/push/PR/merge/tag/deploy;
- no generalized multi-repo automation;
- no massive document split;
- no mutation testing gate.

workflow health, verify gate and diff check pass. Publication remains a later
Alberto-approved action through `scripts/asf_publish_step.ps1`.
